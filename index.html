Preencha os dados do paciente à esquerda e clique em "Gerar Plano Terapêutico" para visualizar o plano individualizado aqui.

O sistema utilizará protocolos baseados em diretrizes internacionais (Surviving Sepsis Campaign, AMIB, etc.) para criar um plano específico para o paciente.

Este sistema é uma ferramenta de apoio à decisão clínica. Todas as intervenções devem ser validadas por médico responsável.
